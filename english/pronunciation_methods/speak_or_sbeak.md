# 送气和不送气
| speak            | sbeak            |
| ---------------- | ---------------- |
| peak /pi:k/      | speak  /spi:k/   |
| tick /tIk/       | stick  /stIk/    |
| cool /ku:l/      | school /sku:l/   |

英语当中送气（aspiration）的音： /p/ /t/ /k/    
tape /teIp/    
pay /pei/    
colour /ˈkʌlə/    

在以下情况/p, t, k/是不送气的： 在重读音节中    
/s/ + 任意/p, t, k/ + 元音    
/s/ + 任意/p, t, k/ + 任意/w, j, l, r/ + 元音
eg. splash  square
